# 模板

## 标准上下文
	标准上下文即在模板中默认传入的参数

1. config

	`flask.config`
	
2. request
	
	`flask.request`，仅在上下文中可用

3. session

	`flask.session`，仅在上下文中可用
4. g

	`flask.g`，仅在上下文中可用
	
5. url_for()

	`flask.url_for()`函数
	
6. get\_flashed\_messages()
	
	`flask. get_flashed_messages()`函数
